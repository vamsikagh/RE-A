# Research Companion AI

A React-based AI research assistant that provides live assistance and draft analysis for academic writing.

## Features

- **Live Assist Chat**: Find research papers on any topic
- **Draft Analysis**: Get AI feedback on research writing quality
- **Quality Scoring**: Automatic scoring with breakdown (Novelty, Alignment, Coherence, Relevance)
- **Sentence Feedback**: Detailed suggestions for improvement
- **Paper Recommendations**: Relevant research papers from Semantic Scholar API
